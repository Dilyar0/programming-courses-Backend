1. Написать функцию buble_sort принимающую в качестве входящего параметра не отсортированный список
2. Алгоритм функции buble_sort должен сортировать список методом пузырьковой сортировки.
3. Функция buble_sort должна возвращать отсортированный список
4. Написать функцию selection_sort принимающую в качестве входящего параметра не отсортированный список
5. Алгоритм функции selection_sort должен сортировать список методом сортировки выбором.
6. Функция selection_sort должна возвращать отсортированный список
